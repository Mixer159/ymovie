# ymovie

Web client for SC2 and webshare api.

## Run

```
npm ci
npm start
npm start debug
```

## Deploy

1. compile sources `npm start`
2. upload content of `./dist` via ftp/sftp
3. changes automatically available on https://ymovie.streamcinema.cz/
